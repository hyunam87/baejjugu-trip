## 🧳 배쭈꾸의 여행계획
가족이 함께 여행 일정을 만들고, 공유하고, 기록하는 **링크 기반 여행 계획 웹앱**입니다.


### ✨ 주요 특징
- 로그인 없이 링크로 바로 접속
- 수정 시 비밀번호 보호
- 날짜별 여행 일정 관리
- 여행지 자동 추천 (지도·맛집·명소)
- 아이 모드 (공룡 🦖 · 고래 🐋 · 우주 🚀)
- 사진 업로드 & 여행 타임라인
- 여행 후 기록(후기) 작성
- PDF 출력으로 공유 가능


---


## 🛠 기술 스택
- **Frontend**: Next.js + React + Tailwind CSS
- **Backend**: Firebase (Firestore, Storage)
- **Map API**: Google Maps Places API
- **배포**: Vercel
